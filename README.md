<h1 align="center"> Resumo do Projeto </h1>
<h3>Usando <strong>PHP e MySQL</strong>, criei uma página de agendamento com objetivo de auxiliar os agendamentos de Salões, Barbearias e Clínicas Médicas. </h3>
